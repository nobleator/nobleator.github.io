About
Though an engineer by trade, I like to consider myself a Renaissance man. I am currently exploring a burgeoning interest in web development and I'm always eager to learn new things. My aspiration is to be a better software developer than most mathematicians, and better at math than most software developers. In my free time I enjoy playing and watching soccer, conquering the world, and banging my head against the desk until Stack Overflow saves me.
Work

But really I do anything that needs to be done, including munging financial data, writing proposals, or stitching together videos. Projects
And last but not least, here are some of my side projects, things that I have been working on just because I find them interesting:
  • NeighborGood: An extension of the AHP decision analysis tool, with a specific application towards home and neighborhood selection. Follows the same general principle (AHP decision analysis), but includes a data aggregation tool to parse out publicly available data into a usable format. Currently in development, with the data aggregation tool incomplete, as well as some missing desired features like filters and bounds on criteria.
    Built with Python, Express, NodeJS.
  • Race and Property: A data visualization project to examine the relationship between ethnicity and property values. Uses data from the US Census bureau and Redfin and the Mapbox API for display. The initial map is done and viewable, but it is still a work in progress. Plans to migrate to a Jupyter Notebook.
    Built with Python, Javascript.
  • This site: Everything on this site, mostly just because I was curious.
    Hosted with Github Pages.
  • Decision Tool: A single page application to step the user through performing decision analysis using the Analytic Hierarchy Process (AHP). Two options exist, an automatic version using pre-populated sample data and a manual version where the user inputs values by hand. Development paused and work migrated to NeighborGood.
    Built with AngularJS, Express, NodeJS.
