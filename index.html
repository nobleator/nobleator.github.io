Projects
And last but not least, here are some of my side projects, things that I have been working on just because I find them interesting:
  • Decision Tool: A single page application to step the user through performing decision analysis using the Analytic Hierarchy Process (AHP). Two options exist, an automatic version using pre-populated sample data and a manual version where the user inputs values by hand. Development paused and work migrated to NeighborGood. Written with AngularJS, Express, NodeJS.
  • Map Guesser: Multiple-choice quiz game. User selects a map title from list of multiple-choice options, is told if they are correct or not, and at the end is given a final score. Still some features not implemented, like restricting and randomizing the guesses per play (rather than using the whole database), adding difficulty levels, and saving/exporting your score. Written with AngularJS, Express, NodeJS.
  • NeighborGood: An extension of the AHP decision analysis tool, with a specific application towards home and neighborhood selection. Follows the same general principle (AHP decision analysis), but includes a data aggregation tool to parse out publicly available data into a usable format. Currently in development, with the data aggregation tool incomplete, as well as some missing desired features like filters and bounds on criteria. Written with Python, Django.
  • DomFall: Just a twinkle in my eye at this point, but I plan to build a game that mimics the everyone-in-the-same-room mechanic present in Spyfall, combined with some Eurogame card and board game design.
